Transform Connector

How to Launch

Launch the tool from the dedicated menu or with the following command.

import faketools.tools.rig.transform_connector_ui
faketools.tools.rig.transform_connector_ui.show_ui()

Usage

  1. Select nodes you want to operate with the tool. Only transform type nodes are valid.
  2. Select checkboxes for attributes you want to operate.
  3. Press Copy Value (copy value), Connect Value (connect), or Zero Reset (zero reset) buttons to perform operations.
