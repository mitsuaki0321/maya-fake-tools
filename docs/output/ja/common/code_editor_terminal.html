ターミナル

概要

コードエディターのターミナルはスクリプトエディターのターミナル部分を埋め込んでいます。
そのため、基本的な機能はスクリプトエディターのターミナルと同じです。

コンテキストメニュー（右クリックメニュー）には、ターミナル特有の機能がいくつかあります。

機能説明

ターミナルのクリア

ターミナルをクリアするには、ツールバーのゴミ箱アイコンをクリックするか、Ctrl+K ショートカットキーを使用します。
または、コンテキストメニューから Clear Terminal を選択します。

ショートカットキー

ターミナルで使用できる主なショートカットキーは以下の通りです。

ショートカットキー 機能説明
Ctrl+C 選択したテキストをコピー
Ctrl+K ターミナル出力をクリア
Ctrl+MiddleWheel フォントサイズの調整
