# FakeTools Help Documentation

Welcome to the help documentation for "FakeTools", a collection of tools for Maya.

This site provides detailed explanations of how to use each tool and its features.

## How to Find Tools

Tools are organized by category:

- **Rig**: Tools for rigging work
- **Model**: Tools for modeling work
- **Animation**: Tools for animation work
- **Common**: Tools that can be used across all workflows

Click on each tool card to navigate to its detailed usage page.
