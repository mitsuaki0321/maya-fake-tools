# FakeTools ヘルプドキュメント

Maya用のツール集「FakeTools」のヘルプドキュメントへようこそ。

このサイトでは、各ツールの使い方や機能について詳しく説明しています。

## ツールの探し方

ツールはカテゴリー別に整理されています：

- **リグ**: リギング作業に関するツール
- **モデル**: モデリング作業に関するツール
- **アニメーション**: アニメーション作業に関するツール
- **共通**: すべての作業に共通して使えるツール

各ツールカードをクリックすると、詳しい使い方ページに移動します。
